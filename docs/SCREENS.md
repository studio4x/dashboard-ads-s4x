# Screens - Dashboard ADS S4X

## Admin
- **Dashboard (/admin)**: Visão geral da plataforma.
- **Clientes (/admin/clients)**: Gestão de clientes.
- **Dashboards (/admin/dashboards)**: Gestão de dashboards por cliente.
- **Google Sheets (/admin/google-sheets)**: Configuração e sincronização de fontes.
- **Logs de Importação (/admin/import-logs)**: Histórico detalhado de sincronização.

## Client App
- **Meus Dashboards (/app/dashboards)**: Lista de dashboards permitidos para o usuário.
- **Dashboard Detail (/app/dashboards/[id])**: Visão executiva e páginas internas.
