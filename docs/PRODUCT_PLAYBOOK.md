# PRODUCT_PLAYBOOK.md
## Product Playbook — Dashboard ADS S4X

### Proposta de Valor
Plataforma própria de dashboards executivos para tráfego pago, substituindo o Looker Studio com uma experiência visual superior e dados centralizados via Google Sheets.

### Personas
- **Gestor de Tráfego (você):** Precisa criar relatórios profissionais para clientes sem depender de ferramentas genéricas.
- **Cliente Final:** Recebe acesso a dashboards limpos, com insights acionáveis, sem precisar entender de métricas técnicas.

### Diferenciais
- Visual premium e executivo
- Multi-página por cliente
- Dados via Google Sheets (consolidados das APIs)
- Insights automáticos
- Multi-cliente com área admin

### Roadmap
1. Fase 1: MVP com mocks (concluída)
2. Fase 2: Google Sheets API real
3. Fase 3: Auth Supabase
4. Fase 4: Banco real
5. Fase 5: Integrações diretas
